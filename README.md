# Ordinary Serial Port Assistant (OSPA)

一款基于Qt 5.14.2 (MSVC 2017, 32 bit)开发的跨平台串口调试助手。

## 功能特性

- 多标签页窗口，支持同时管理多个串口连接
- 接收区文本搜索功能，支持正则表达式
- YMODEM文件传输功能（目前仅支持上传，不支持接收）
- CRC自动计算功能，支持多种CRC算法模型
- 支持自定义CRC插入位置，-1或留空表示默认值（如首位或末尾）
- 深色/浅色主题切换
- 多语言支持（目前支持中文和英文）

## 开发说明

本项目设计灵感来源：
- XCOM - 参考其功能布局与界面设计
- SuperCom - 参考其CRC校验功能与多标签页功能实现
